# Raw Dataset

The dataset used in this project is the **Philippine Labor Force Survey (April 2016) Public Use File**.

Due to GitHub file size limitations, the dataset is not included in this repository.

To reproduce the analysis:

1. Download the dataset from the **Philippine Statistics Authority (PSA)**.
2. Place the CSV file in this folder.

Expected file location:

```
data/raw/LFS_PUF_April_2016.csv
```

Once the dataset is placed in this directory, the notebook will run without modification.
